# GameVault

Полноценное Flask-приложение для отслеживания игр через RAWG API. Интерфейс самого сайта полностью на английском языке.

## Что внутри

- Главная страница с популярными играми RAWG.
- Поиск игр по названию.
- Страница игры с описанием, рейтингом, жанрами, датой релиза и платформами.
- Статусы `Playing`, `Wishlist`, `Completed`.
- Регистрация, вход и выход через `Flask-Login`.
- Хеширование паролей через Werkzeug.
- SQLite база `games.db`.
- Персональная библиотека для каждого пользователя.

## Запуск

1. Создай и активируй виртуальное окружение:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Установи зависимости:

```powershell
pip install -r requirements.txt
```

3. Запусти приложение:

```powershell
python app.py
```

4. Открой сайт:

```text
http://127.0.0.1:5000
```

## RAWG API

В `app.py` уже добавлен твой RAWG API key, поэтому проект запускается сразу. При желании можно переопределить ключ через переменную окружения:

```powershell
$env:RAWG_API_KEY="your_rawg_key"
python app.py
```

## База данных

Файл `games.db` создается автоматически при запуске приложения.

Таблица `users`:

- `id`
- `username`
- `email`
- `password_hash`

Таблица `user_games`:

- `id`
- `user_id`
- `game_id`
- `status`

В базе хранятся только пользователь, RAWG `game_id` и статус. Все данные об играх загружаются из RAWG API.
